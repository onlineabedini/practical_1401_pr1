## how to start?
step1 : install nodejs and vuecli and nuxtcli
step2 : install git 
step3 : clone project
step4 : run applications using following commands: 


**install dependencies:**
```
npm install
```

**nodejs app**
```
npm start
```

**nuxtjs**
```
npm run dev
```
